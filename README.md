# CVPR 2026 Children Challenge Reproducible Runs

This repository contains reproducible experiment artifacts for the Kaggle competition:

https://www.kaggle.com/competitions/cvpr-2026-the-first-ai-children-challenge

The included candidates use released competition labels and deterministic light features extracted from the provided keypoint data. Candidate CSV files are paired with manifest files that record input and output SHA256 hashes.

## Contents

- `data/raw/`: released Track 1 and Track 2 training labels.
- `features/patient_light_features.csv`: patient-level summary features used by the official-data experiments.
- `scripts/`: deterministic candidate generation and validation scripts.
- `submissions/candidates/`: generated candidate CSV files and manifests.
- `submissions/submission_results_*.csv`: public leaderboard results for submitted candidates.
- `docs/`: experiment summaries.

## Reproduce Candidates

Install the small Python dependency set:

```bash
python -m pip install -r requirements.txt
```

Regenerate the May 14 targeted candidates:

```bash
python scripts/generate_2026_05_14_targeted_candidates.py \
  --output-dir submissions/candidates/2026-05-14-targeted
```

Validate the best candidate:

```bash
python scripts/validate_submission.py \
  submissions/candidates/2026-05-14-targeted/m01_k08_t1_72_meta9_thr060.csv
```

Expected output includes:

```text
sha256=2dae0cb702427e1e21b5ca97419ff48a492dad24bfdff0ae49ac2e9eb607b566
status=pass
```

The May 14 loop found that reducing Track1-72 by one bit from the previous
best improves the reproducible public score to `0.70047`.

## Notes

The full extracted keypoint dataset is not included because it is large. The included `patient_light_features.csv` is the deterministic feature table used by the submitted official-data candidate generators.

Some higher public-score diagnostic experiments are intentionally excluded from this package because this repository is focused on reproducible official-data candidates.
